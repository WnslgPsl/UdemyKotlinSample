UdemyKotlinSample
==================

해당 Repository는 [Udemy Kotlin 강좌][udemy]를 듣고 작성한 Sample Repository 입니다.

해당 강의를 청강 한 후, 개인적인 생각도 반영된 샘플 코드이므로, 기능 또는 코드 일부가 다를 수 있습니다.

Kotlin을 이용한 Android MVP Pattern 구현에 대한 샘플을 작성한 코드 입니다.
